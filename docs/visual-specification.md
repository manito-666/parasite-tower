# 寄生魔塔 - 视觉规范文档
# 生物朋克 × 新艺术运动 (Biopunk × Art Nouveau)

## 一、核心风格定位

**生物朋克 × 新艺术运动**: Canvas 2D 实时渲染，以有机腔体建筑、发光菌膜与虹彩甲壳为视觉语言

**色彩基调（生物荧光调色板）**:
- 深空底色: `#0d0818`（深紫黑）/ `#080514`（近黑）
- 生物青: `#00ffd0`（荧光膜冷光）
- 生物洋红: `#ff006e`（细胞核活性色）
- 生物紫: `#b455ff` / `#8844ff`（神经脉冲）
- 环境紫: `#2d1b4e`（菌丝层）

**视觉层次**:
1. 背景层: 有机腔体瓷砖（紫调石材 + 菌膜覆盖纹理）
2. 地形层: 半透明膜质墙体 / 括约肌下行楼梯 / 紫色穹顶上行楼梯
3. 实体层: 生物发光怪物 / 生物朋克玩家 / 荧光道具
4. 特效层: 虹彩粒子 / 菌膜脉冲 / 血管贝塞尔曲线

## 二、基础设计规范

### 2.1 网格系统
```
外轮廓: 30×30 (留1px边距)
核心区域: 24×24 (中心8×8为视觉焦点)
细节层: 最小可识别单位 2×2px
瓷砖尺寸: T×T (运行时 canvas 像素)
```

### 2.2 色板规范（生物荧光）
每个精灵使用以下色系：
- 基础色: 紫黑底 `#0d0818` / `#1a0e2e`
- 发光色: 生物青 `#00ffd0` / 生物洋红 `#ff006e` / 神经紫 `#b455ff`
- 高光色: 虹彩 `hsla(H, 100%, 70%, α)` 随时间变化色相
- 菌膜: `rgba(0,255,208, 0.04~0.12)` 半透明叠加

### 2.3 动态发光技术
使用 Canvas 2D `shadowColor` + `shadowBlur` 实现生物发光:
```javascript
// 荧光轮廓
ctx.shadowColor = '#00ffd0';
ctx.shadowBlur = 8;

// 虹彩翅膜（随时间偏移色相）
const iridHue = (Date.now() / 30) % 360;
ctx.fillStyle = `hsla(${iridHue}, 100%, 70%, 0.08)`;

// 神经血管贝塞尔曲线
ctx.strokeStyle = 'rgba(0,255,208,0.2)';
ctx.bezierCurveTo(...);
```

## 五、地形渲染规范

### 5.1 墙体（有机腔体壁）
```javascript
// 基础色: 紫调石材
rgb(26 + v*2, 14 + v*2, 46 + v*3)

// 勾缝: rgba(13,8,24,0.7)

// 菌膜呼吸叠加
const membraneAlpha = 0.04 + 0.02*Math.sin(now/1200 + x*0.7 + y*0.5);
sc.fillStyle = `rgba(0,255,208,${membraneAlpha})`;

// 血管纹（每3格一条贝塞尔曲线）
if ((x+y) % 3 === 0) {
  sc.strokeStyle = 'rgba(180,85,255,0.08)'; sc.lineWidth = 0.5;
  sc.quadraticCurveTo(...); // 水平穿越弧
}

// 噪声: rgba(180,85,255,...) 紫色孢子斑点
// 生物苔藓: rgba(0,255,208,...) 荧光青替代传统绿
```

## 八、动画规范（生物律动）

### 8.1 核心动画参数
```javascript
// 生物脉冲（全局基准）
const pulse = Math.sin(Date.now() / 700 + phase) * 0.04 + 0.96;

// 虹彩偏移（翅膜/拖尾）
const iridHue = (Date.now() / 30) % 360;

// 菌膜呼吸（墙体叠加）
const membraneAlpha = 0.04 + 0.02 * Math.sin(now / 1200 + x*0.7 + y*0.5);

// 相位环旋转（T4怪物）
const ringHue = (Date.now() / 20 + i * 30) % 360;
```

### 8.2 附身动画（意识侵入序列）
```
总时长: 2.5秒

0.0-0.5s: 接触 — 荧光触须贝塞尔伸展
0.5-1.5s: 侵蚀 — HP条 #00ffd0→#b455ff→#080514 变色，成功率浮动
1.5-2.0s: 崩解 — 像素化为 rgba(0,255,208,...) 孢子粒子
2.0-2.5s: 重组 — 生物质凝聚，虹彩轮廓扩散，形态确立

失败(0.8s):
  触须断裂 → 生物洋红闪光 → 玩家硬直
```

### 8.3 污染特效（神经脉冲粒子）
```
颜色: 按区域荧光色（生物青/神经紫/细胞洋红）
形状: 圆点 2px
轨迹: 向上漂浮 + 随机水平偏移
寿命: 0.5-1.0s
淡出: alpha 1.0→0.0
```

### 8.4 孢子粒子（替代旧版火焰）
```
生物孢子: rgba(0,255,208,0.5) 荧光青
轨迹: 螺旋上浮
寿命: 0.8s
粒子数: 与污染值正相关
```

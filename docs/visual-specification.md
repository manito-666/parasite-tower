# 寄生魔塔 - 视觉规范文档

## 一、核心风格定位

**像素艺术风格**: 32×32 像素，低饱和度暗黑科幻，强调污染感与生物异化

**色彩基调**:
- 主色调: 深灰 (#2a2a2a) + 紫红污染色 (#8b3a8b)
- 辅助色: 生物质绿 (#4a7c59)、锈蚀橙 (#d97742)、冰冷蓝 (#4a6b8a)

**视觉层次**:
1. 背景层: 实验室瓷砖/工业金属纹理
2. 地形层: 墙体/门/楼梯
3. 实体层: 怪物/玩家/道具
4. 特效层: 污染雾气/战斗粒子/寄生动画

---

## 二、32×32 像素规范

### 2.1 网格系统
```
外轮廓: 30×30 (留1px边距)
核心区域: 24×24 (中心8×8为视觉焦点)
细节层: 最小可识别单位 2×2px
```

### 2.2 色板限制
每个精灵最多使用 **8 种颜色** (含透明):
- 基础色 (1色)
- 阴影色 (2-3色，逐级加深)
- 高光色 (1-2色)
- 特征色 (1-2色，如眼睛/污染纹理)

### 2.3 抖动技术
使用 **2×2 棋盘抖动** 模拟渐变:
```
明→暗过渡:
██░░  →  ░░██  →  ░░░░
██░░     ░░██     ░░░░
```

## 三、怪物设计规范

### 3.1 区域1: 实验室 (1-10层)
**色调**: 灰白 + 血红
**怪物特征**: 实验动物变异，保留原始生物轮廓

**实验鼠** (rat):
```
尺寸: 16×12px (小型)
关键帧:
  - 待机: 蜷缩，尾巴微动 (2帧循环)
  - 攻击: 前扑，露出獠牙 (3帧)
  - 死亡: 侧倒，血迹扩散 (4帧)
色板: #8b4513(棕), #654321(深棕), #ff0000(血), #ffffff(牙)
```

**畸变狼** (wolf):
```
尺寸: 28×24px (中型)
关键帧:
  - 待机: 低吼，脊背起伏 (3帧)
  - 攻击: 跃起扑咬 (5帧)
  - 狂暴: 全身发红，速度×2 (HP<50%触发)
色板: #8b0000(暗红), #4a0000(深红), #ffff00(狂暴眼), #2a2a2a(阴影)
特效: 狂暴时周身红色粒子
```

### 3.2 区域2: 培育区 (11-20层)
**色调**: 绿色生物质 + 黄色脓液
**怪物特征**: 昆虫/软体动物，多节肢/触手

**蟑螂群** (roach):
```
尺寸: 12×8px (极小，3-5只成群)
动画: 快速爬行 (4帧，60fps)
色板: #654321(壳), #8b7355(腹), #000000(触角)
群体AI: 包围玩家，分散攻击
```

**黏液怪** (slime):
```
尺寸: 24×20px (可变形)
关键帧:
  - 待机: 蠕动，半透明 (4帧)
  - 攻击: 伸出伪足 (3帧)
  - 分裂: 死亡时分裂成2个小史莱姆 (特性)
色板: #32cd32(主体), #228b22(深绿), #90ee90(高光), alpha=0.7
特效: 移动时留下绿色粘液轨迹
```

### 3.3 区域3: 污染核心 (21-30层)
**色调**: 紫红污染 + 黑色腐蚀
**怪物特征**: 严重畸变，多眼/多肢/融合体

**融合体** (fusion):
```
尺寸: 32×32px (大型，占满格子)
构成: 3-4种生物器官拼接
关键帧:
  - 待机: 器官独立蠕动 (5帧)
  - 攻击: 随机器官突刺 (6帧)
  - 再生: 受伤部位快速愈合 (特性)
色板: #8b3a8b(污染紫), #4a0000(腐肉), #00ff00(异常眼), #000000(缝合线)
特效: 周身紫色雾气，每秒扩散1格
```

**污染行者** (walker):
```
尺寸: 28×30px (人形)
特征: 前科学家，身体被污染侵蚀
关键帧:
  - 待机: 痛苦挣扎 (4帧)
  - 攻击: 污染触手鞭打 (5帧)
  - 死亡: 身体崩解成紫色粒子 (8帧)
色板: #2a2a2a(残余衣物), #8b3a8b(污染), #ff00ff(触手), #ffffff(骨骼)
```

### 3.4 区域4: 深渊 (31-40层)
**色调**: 深蓝黑 + 冰冷白
**怪物特征**: 非欧几何，违反物理规则

**虚空潜伏者** (lurker):
```
尺寸: 32×32px (不定形)
特征: 半透明，边缘模糊
关键帧:
  - 待机: 在虚空中闪烁 (6帧，alpha 0.3-0.8)
  - 攻击: 瞬移到玩家身后 (传送特效)
  - 相位: 50%概率闪避物理攻击
色板: #1a1a2e(深蓝黑), #4a6b8a(冰蓝), #ffffff(眼), alpha渐变
特效: 移动时空间扭曲波纹
```

### 3.5 区域5: 终焉 (41-50层)
**色调**: 全黑 + 血红高光
**怪物特征**: BOSS级，多阶段变化

**原初寄生体** (origin):
```
尺寸: 64×64px (超大，占4格)
阶段1 (HP 100%-60%):
  - 形态: 巨大肉瘤，多触手
  - 攻击: 范围鞭打 (8帧)
阶段2 (HP 60%-30%):
  - 形态: 裂开露出核心
  - 攻击: 召唤小怪 + 污染射线
阶段3 (HP 30%-0%):
  - 形态: 核心暴露，狂暴
  - 攻击: 全屏污染爆发
色板: #4a0000(肉), #8b3a8b(污染), #ff0000(核心), #000000(深渊)
特效: 死亡时屏幕震动，紫色爆炸扩散全屏
```

## 四、玩家角色设计

### 4.1 基础形态 (未寄生)
```
尺寸: 20×20px
外观: 紫色半透明球体，中心"寄"字
动画:
  - 待机: 轻微脉动 (2帧)
  - 移动: 拖尾残影 (3帧)
色板: #8b3a8b(主体), #ff00ff(高光), alpha=0.8
```

### 4.2 寄生形态 (继承怪物外观)
```
规则: 使用当前寄生怪物的精灵，叠加紫色污染纹理
污染纹理: 2×2 紫色斑点，覆盖率 = 污染值%
  - 0-30: 无纹理
  - 31-60: 10%覆盖
  - 61-80: 30%覆盖
  - 81-100: 60%覆盖，眼睛发紫光
```

## 六、动画规范

### 6.1 帧率标准
- 待机动画: 2-4帧，15fps
- 移动动画: 3-4帧，30fps
- 攻击动画: 4-6帧，30fps
- 特效动画: 6-12帧，60fps

### 6.2 寄生动画序列
```
总时长: 1.5秒 (45帧 @ 30fps)

帧1-10: 玩家球体冲向怪物
帧11-20: 球体钻入怪物体内 (怪物闪烁)
帧21-30: 怪物剧烈抖动，紫色纹理扩散
帧31-40: 怪物眼睛变紫，停止抖动
帧41-45: 紫色光环扩散，寄生完成

失败动画 (15帧):
帧1-10: 球体被弹开
帧11-15: 球体恢复原位，怪物反击
```

### 6.3 污染特效
```
粒子系统:
  - 发射速率: 污染值 / 10 个/秒
  - 粒子寿命: 0.5-1秒
  - 颜色: #8b3a8b → #ff00ff (渐变)
  - 运动: 向上漂浮 + 随机横向偏移
  - 大小: 2×2px → 1×1px (缩小)
  - alpha: 1.0 → 0.0 (淡出)
```

## 七、Shader伪代码

### 7.1 污染扭曲效果 (污染>60时)
```glsl
// 屏幕空间扭曲
vec2 distort(vec2 uv, float pollution) {
    float strength = (pollution - 60.0) / 40.0; // 0-1
    float time = u_time * 0.5;

    vec2 offset = vec2(
        sin(uv.y * 10.0 + time) * 0.01,
        cos(uv.x * 10.0 + time) * 0.01
    ) * strength;

    return uv + offset;
}

// 紫色叠加
vec4 applyPollution(vec4 color, float pollution) {
    float intensity = pollution / 100.0;
    vec3 pollutionColor = vec3(0.545, 0.227, 0.545); // #8b3a8b

    color.rgb = mix(color.rgb, pollutionColor, intensity * 0.3);
    return color;
}
```

### 7.2 虚空潜伏者相位效果
```glsl
// 边缘溶解
float edgeFade(vec2 uv, vec2 center, float radius) {
    float dist = distance(uv, center);
    float noise = fract(sin(dot(uv, vec2(12.9898, 78.233))) * 43758.5453);

    return smoothstep(radius, radius * 0.8, dist + noise * 0.1);
}

// 闪烁
float flicker(float time) {
    return 0.3 + 0.5 * sin(time * 3.0) + 0.2 * sin(time * 7.0);
}
```

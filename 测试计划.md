# Parasite Tower 测试计划

## P0 — 致命/阻塞性Bug（必须通过才能发布）

### T-P0-01: 游戏启动流程
- [ ] 冷启动APP → 显示开始界面 → 点击开始 → 职业选择界面出现
- [ ] 选择泰坦/幽灵/虫群，长按确认 → 进入游戏画面，角色出现在地图上
- [ ] 退出APP后重新打开 → 继续游戏按钮可用 → 点击后恢复存档位置/状态

### T-P0-02: 移动系统
- [ ] 四方向按钮均可移动角色
- [ ] 角色不能穿墙（tile=0）
- [ ] 走到出口（tile=2）→ 进入下一层

### T-P0-03: 战斗核心
- [ ] 接触怪物 → 弹出遭遇面板（顶部悬浮）+ 底部按钮（攻击/防御/附身/逃跑）可点击
- [ ] 点击攻击 → 怪物掉血，战斗日志显示伤害数字
- [ ] 怪物血量归零 → 战斗结束，遭遇面板关闭，获得EP
- [ ] 玩家血量归零 → 触发死亡/形态切换界面
- [ ] 逃跑按钮 → 关闭战斗界面，角色可继续移动

### T-P0-04: 附身/寄生
- [ ] 点击附身 → 打开谈判界面（威压/交易/共鸣三选项）
- [ ] 选择后 → 判定成功/失败
- [ ] 附身成功 → 继承怪物属性，战斗界面关闭，**日志不闪烁**
- [ ] 附身成功后 → 污染值增加，top-bar属性刷新为新形态

### T-P0-05: 存档/读档
- [ ] 移动几步后手动退出 → 重新进入 → 位置/HP/ATK/DEF/EP/污染值均恢复
- [ ] 跨层存档 → 楼层号、怪物状态正确恢复
- [ ] 删除存档 → 确认后存档消失，重新开始

### T-P0-06: UI基础
- [ ] top-bar显示：HP条、数值HP/MaxHP、ATK、DEF、污染%、楼层号 — 均为**真实数字**（非???，除非哑剧签名且楼层≥10）
- [ ] 遭遇面板按钮文字为中文（战斗/逃跑/附身/长按2秒）
- [ ] 消息面板正常滚动，不溢出屏幕

## P1 — 重要功能验证（影响核心体验）

### T-P1-01: 防御机制
- [ ] 点击防御 → 恢复8%MaxHP + 受伤减半，日志显示正确
- [ ] 防御10次 → 解锁成就"铁壁"

### T-P1-02: 形态系统
- [ ] 附身新怪物后 → 形态栏显示新形态
- [ ] 点击形态栏切换 → 属性变为目标形态，提示重组僵直1回合
- [ ] 当前形态死亡 → 弹出形态选择界面 → 切换到存活形态
- [ ] 所有形态死亡 → 游戏结束或崩溃触发

### T-P1-03: 污染技能树
- [ ] 污染达50% → 消息提示"感官剥离"技能解锁
- [ ] 污染达60% → 消息提示"暴击增幅"被动解锁
- [ ] 污染达70% → 消息提示"现实扭曲"技能解锁
- [ ] 污染达75% → 消息提示"污染护盾"被动解锁
- [ ] 污染达80% → 消息提示"预知闪回"技能解锁
- [ ] 污染达90% → 消息提示"真实视界"被动解锁 + 视野明显扩大
- [ ] 点击污染值 → 打开污染技能面板，已解锁技能可使用
- [ ] 使用感官剥离 → 10步内免疫地形伤害，10步后提示效果结束
- [ ] 使用任意主动技能 → 锁定显示"5层"，进新层递减

### T-P1-04: 职业能力
- [ ] 泰坦：泰坦之怒按钮可用 → 点击后HP×1.5, ATK/DEF+15, 持续10回合
- [ ] 幽灵：虚空行者 → 5回合无敌
- [ ] 虫群：虫群之心 → 生成分身

### T-P1-05: 进化/EP系统
- [ ] 击杀怪物/附身 → 获得EP
- [ ] 菜单中可查看进化路径
- [ ] 花费EP购买进化 → 效果立即生效（如减伤、吸血等）

### T-P1-06: 楼层签名
- [ ] F2-F9层：不出现哑剧/黑暗降临/时间加速签名
- [ ] 进入有签名的楼层 → 显示签名横幅提示
- [ ] 签名效果生效（如再生之地每步回血、毒雾每步加污染等）

### T-P1-07: 自动战斗
- [ ] 长按战斗按钮1秒 → 按钮变为"自动战斗..."，自动连续攻击
- [ ] 怪物死亡 → 自动战斗停止
- [ ] 松手 → 自动战斗取消

### T-P1-08: 怪物HP条
- [ ] 地图上怪物头顶HP条为单层清晰显示（非2层堆叠）
- [ ] 怪物受伤后HP条缩短，颜色变化（绿→黄→红）

## P2 — 次要功能/边缘场景

### T-P2-01: 特殊楼层
- [ ] F5镜像实验室 → 特殊战斗机制触发
- [ ] F25净化仪式 → 选择界面出现
- [ ] F50起源之地 → 结局选项出现

### T-P2-02: 污染效果
- [ ] 污染31-60% → 地图出现幻觉怪物（走近消失）

### T-P2-04: 成就系统
- [ ] 满足条件 → 成就横幅弹出
- [ ] 菜单中可查看已解锁成就

### T-P2-05: 锚点系统
- [ ] 手动创建锚点 → 保存当前状态
- [ ] 死亡时可选择回滚到锚点

### T-P2-06: 碎片/技能合成
- [ ] 击杀怪物获得碎片
- [ ] 收集足够碎片 → 合成主动技能
- [ ] 战斗中使用技能 → 效果生效

### T-P2-07: 羁绊系统

- [ ] 达到等级2 → 解锁被动技能

### T-P2-08: 商店系统

- [ ] 消耗EP购买 → 效果生效（回血/加属性/减污染等）

### T-P2-09: 音频
- [ ] 战斗/附身/死亡 → 播放对应音效
- [ ] 不同楼层 → 背景音乐变化

### T-P2-10: 边缘场景
- [ ] 连续快速点击攻击按钮 → 不卡死/不重复计算
